Configuration file

params:
  username: "richy2509"
  projectName: "templator"
templateDir: "/Users/XXX/workspace/templator-model" 
modelfile: "model.ftl" 
outputfile: "/Users/XXX/workspace/templator-model/out.ftl"
  • templateDir : Path folder where models are stored
  • modelfile : Relative path to the template file from the "templateDir" folder
  • outputfile : Absolute path to the output file

Model file

<html>

  <body>
